# 今日运势生成器

基于jrys-max开发，原本只是想修复一个bug，但是写着写着就一发不可收拾了（

## 功能

- [x] 获取每日运势
- [x] 支持 Monetary
- [x] 自定义背景图
- [x] 随机一言
- [x] 可自定义的搞怪黄历
- [x] 自定义签到等级
- [x] 好看！
